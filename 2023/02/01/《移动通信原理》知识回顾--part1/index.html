《移动通信原理》知识总结--part1

发表于 2023-02-01 更新于 2023-02-02 分类于 移动通信原理

chapter1 移动通信发展历程

第一代通信系统–话音业务需求：从无到有（20世纪80年代）

  1. 1G两种制式AMPS（北美）、TACS（欧洲）分别占用频段800MHz、900MHz。
  2. 1G中引入了双工的工作模式。特殊地，两端同时可以进行通信是双工通信，若不能同时进行则是半双工通信（分时的双工通信）。
  3. 下行链路dl：基站发，移动台收。上行链路：移动台发，基站收。使用载波频率区分，代表频分双工fdd–1G中采用的方式。tdd使用时间的方式进行上下行区分–我国3G、4G中商用使用的模式。
  4. 采用模拟电路处理模拟信号的方式进行。
  5. 采用宏蜂窝小区：大区制–覆盖范围大、但不能适应用户数量的增加

第二代通信系统–话音技术需要：数字信号处理技术提高质量（20世纪90年代）

  1. 数字化的处理技术：数字信号处理、业务数字处理、信令数字处理、高效调制、信道均衡、多信道并行等技术。数字化技术便于实现小型化，集成度提高。
  2. 组网技术采用宏蜂窝、小区、微区、微小区的多重覆盖方式。
  3. 主要制式：GSM（区分用户即多址接入技术–TACS、AMPS模拟制中采用频分多址技术FDMA）采用TDMA的多址接入技术，北美提出的CDMA中典型的IS95采用CDMA码分多址
  4. GSM双工方式：FDD，采用TDMA的多址接入技术，核心频段兼容TACS，采用900MHz+1800MHz（扩展频段，改成DCS–digital cell system）
  5. 北美D-AMPS：digital AMPS，兼容AMPS的800MHz，采用TDMA的多址接入技术，FDD的双工方式
  6. 日本PDC：采用TDMA的多址接入技术，简化为PHS，进一步简化为PAS，之后引入中国，称作小灵通
  7. 北美CDMA：采用CDMA的多址接入技术，兼容AMPS的800MHz，FDD的双工方式

第三代通信系统–移动数据业务需求

  1. 采用自适应技术：根据移动信道的动态特征，采用自适应调制、自适应编码、自适应接入、网络自适应等来适应信道的变化。
  2. 网络技术在1G、2G主要采用电路交换技术来支持语音业务，3G中采用分组（IP）交换技术来支持数据业务
  3. 3G中支持多媒体业务，因此在业务技术方面的业务分类、编码组帧、数据压缩等进行了技术创新
  4. 技术体制：WCDMA（3GPP制定）-联通，cdma2000（3GPP2制定）–电信，TD-SCDMA（中国提出，后加入3GPP标准化）–移动
  5. WCDMA：FDD的CDMA制式，带宽一个载波5MHz、码元速率3.84MHz、工作频段–上行1920-1980MHz（60MHz）、下行2110-2170MHz（60MHz）
  6. cdma2000：IS95的演进，带宽1.25MHz、工作频段：800MHz（与AMPS兼容），FDD的双工方式
  7. TD-SCDMA：TDD的双工方式，S代表智能天线、同步、多址接入采用CDMA的方式，一个载波带宽1.6MHz、码元速率1.28MHz、工作频段较多
  8. 针对不同业务，采用不同QOS要求：误码率、时延E2E（主叫到被叫的端到端时延），划分为会话类业务（实时性要求高150-200ms，差错要求较低1e-3）、流业务(视频播放,误码率1e-4)、交互类业务（ftp、微信、微博,误码率1e-5）、后台业务（email,误码率1e-6）

第四代通信系统–移动数据质量需求

  1. 主要制式：LTE（WCDMA的演进、CDMA2000没有设备支持）–联通、电信、移动
  2. wibro由IEEE演进推动，典型的制式有wimax（802.16m），也是4G的候选方案，但是普及率很低
  3. LTE：long term evolution，还可以细分FDD-LTE（电信、联通）、TDD-LTE（移动）。采用新型的信号处理技术实现高校传输以提高频谱效率、实现高效覆盖以提高功率效率、实现高校接入以提高网络效率。高速移动100Mbps-1Gbps，高频谱效率–单位时间、单位频带内所发送的信息量：10-20bit/HZ/S，高信息量–各种多媒体业务都支持
  4. 参数：载波带宽可以灵活配置1.25M、2.5M、5M、10M、15M、20MHz（带宽增加，传输速率越多）、多址接入方式–下行OFDMA（正交频分多址接入技术，采用数字正交化技术）&上行SC-FDMA（单载波频分多址接入技术）

第五代通信系统–移动通信技术+新型技术融合

  1. 5G为用户提供无缝衔接、高速传输、高满意度的业务服务
  2. 技术指标：频谱效率相比4G提高10倍；网络结果采用新型（分布式网络结构）和智能化技术使系统吞吐量提高25倍以上；挖掘新的频段资源（高频段、毫米波、可见光）
  3. 数字指标：单位面积数据量提升1000倍、传输速率达到10Gbps、端到端时延缩短5倍，可联网设备增加10-1000倍。
  4. 应用场景：eMBB增强移动宽带（商业区、住宅区用户数支持高密度用户）、mMTC大连接通信网络（机器通信、传感器间的通信–物流）、uRLLC高可靠性低时延通信（自动驾驶）
  5. 3GPP、IEEE制定技术标准，之后政府部门ITU进行发布
  6. 关键技术：大规模天线massive mimo基站端64根以上天线、新型多址接入技术NOMA、超密集组网（超小蜂窝）、全频谱接入、多载波技术（增强OFDM技术）、先进的编码调制技术（逼近香农容量限）
  7. 物理层传输技术：256QAM、1024QAM、NOMA（SCMA–新一代多址接入方式）、Massive mimo（128-256天线阵列、3D-MIMO）、LDPC（data）、POLAR码（command 可靠性更好）、Turbo接受技术（优化接收机迭代接受以优化通信整体）、高频段传输（3GHz以上的频段，频谱高、带宽宽）、同时同频的全双工
  8. 网络层技术：D2D技术（允许终端直接通信）、超密集组网、cloud-ran网络架构云接入网络架构

未来移动通信技术-6G

  1. AI+通信：AI算法是人类的超级助手，AI成为人类的超级代理进行决策，AI是人类的君王。在通信中，AI定义通信网络：ADN，具有全知全能功能。
  2. 量子无线通信

chapter2 无线传播与移动信道

移动信道的特点

  1. 移动通信信道的三个特点：传播的开放性、接受地点地理环境的复杂性和多样性、通信用户的随机动移动。信号在移动信道传播中可以归纳为3类损耗和4中效应。
  2. 3类损耗：路径传播损耗（大尺度损耗衰落）、慢（中等尺度）衰落损耗（阴影衰落：电磁波场强的中值随时间缓慢变化–阴影区域）、快衰落（小尺度衰落）损耗–进一步可以划分为空间选择性、时间选择性、频率选择性衰落
  3. 4种效应：
    （1）阴影效应（大型建筑物的遮挡，只能衍射）：移动台在阴影区域可能会出现中断的现象（2）远近效应：多个用户的相互干扰。基站收到近端用户的功率高，也称作MAI–多址干扰。（3）多径效应：接收端接受发送端的多条路径的信号，也称作MPI–多径干扰，就会导致码间干扰ISI。（4）多普勒效应：接收端运动导致的多普勒效应，静止、室内场景下不考虑。
  4. 移动通信中影响性能的主要噪声和干扰：加性正态白噪声、多径干扰MPI、多址干扰MAI

三种主要的快衰落–通信场景中一般同时存在

  1. 空间选择性衰落：基站发送信号时由于角度扩展会形成波束，接收端信号的角度谱展宽，不同接收机的位置不同，接收信号的强度也不同。一般采用空间分集–使用两根接收天线来改善信号质量。
  2. 频率选择性衰落（多径）：时域上发送冲激信号，输出信号频率上的信号不在平坦。峰峰之间间距–相干带宽B，时域上信号会出现展宽，时间的最大扩展成为最大多径时延T，B=1/T。如果两个频率分量的频率差超过相干带宽，则认为两个信号的统计特性相互独立，否则频率分量有相关性，频率选择性就是代表了频谱上的分量同时增加或减效。
  3. 时间选择性衰落（多普勒）：频域上发送冲激信号，输出信号时间上不是一个规整的正弦波，对应频率上会出现频域展宽，最大的频率展宽称为最大多普勒频移f，时间上会定义相干时间t：采出的样值，其时间差如果大于相干时间，则这两个样值衰落相互独立，否则具有相关性，其中，t≈1/f。
  4. 信号带宽W、周期Ts，信道相干带宽B、相干时间Tc的关系：（1）W>>B，信号为宽带信号，称为典型的频率选择性衰落.W<<B，信号的衰落有相关性，称为平坦衰落（2）Ts>>Tc:快衰落–信号的一个符号周期会起伏很多次，Ts<<Tc：慢衰落。
  5. 1G W=25KHz，B=200KHz TACS为平坦衰落；3G W=5MHz，B=200KHz 频率选择型衰落。3G往后一般是频率选择性衰落。
  6. 通常在慢速移动的场景下，典型的一般是慢衰落。
  7. 时频双选信道：时间、频率选择性衰落，高速运动场景下会出现这样的现象，信号受损最严重
  8. 平坦快衰落：卫星信号，卫星快速移动，但是一般和地面是直射传播信号（多径较少）
  9. 平坦慢衰落：固定接入点会出现，一般情况很少出现。
  10. 都市环境中角度扩散为20°，时延扩展为5us–对应的相干带宽为200KHz，多普勒频移120HZ–相干时间10ms—>用于帧格式的设计如一帧的时长需要小于相干时间。

传播类型与信道模型的定量分析

  1. 大尺度路径损耗的规律：距离（真空环境中功率随距离呈平方衰减，恶劣情况可以达到5.5）、载波频率（f越高，信号衰减越大；高频段覆盖相同的地理区域需要的基站数越多）。典型的模型：hata模型、okumura-hata模型（这2种模型常用于网络的发展和规划、通信体制和标准的设定）、wim模型、室内传播模型
  2. 中等尺度衰落（阴影衰落）：接收信号场强中值随着时间缓慢变化，服从的分布为对数正态分布。
  3. 小尺度衰落中的多普勒效应导致信号的相关谱–自相关函数对应的功率谱：clarke模型–u型谱（典型谱）
  4. 3种损耗结合具体场景归纳为：高速宽带车载移动信道、宽带慢速移动信道、宽带高速并且具有智能天线的信道、窄带高速移动车载信道、低速窄带步行信道
  5. 信道的统计特征、随机样值的概率分布特征：瑞利信道（发射机、接收机均经过多径散射、没有直射信道）、Rice分布（经过多径散射、包含直射信道）、Gausse分布（直射信道最强，电子通信设备的噪声产生高斯噪声）。瑞利最差，莱斯中等、高斯分布最好。nakagami分布是带着参数的分布，涵盖三种分布，常用于理论分析。

无线信道模型–多径、多普勒、白噪声

  1. 单径衰落信道：功率谱常用的有三种–平坦谱、高斯功率谱、典型功率谱，信号的幅度服从瑞利分布，信号的相位服从高斯分布。具体产生的方法包括时域卷积（功率约束的信号经过低通滤波器产生）、频域相乘（带宽越窄，fft、ifft点数越高）、jakes算法来产生（确定性信道建模的方法–随机初始化相位的正弦波叠加而成，中心极限定理知其I、Q路服从高斯分布）
  2. 多径衰落信道：多径信道采用WSSUS（广义平稳非相干散射）模型来表示，各路都是单径衰落信道模型。
  3. 多天线模型–MIMO模型：分析模型（MNL条路径，wss_us的矢量版本）、物理模型（基于散射体构建–空时频三个维度刻画相应的随机行为、统计特征。jakes模型的推广-3GPP标准）

chapter3 多址技术和扩频通信

多址技术的基本概念

  1. multiaccess与multiplexing都属于信号的正交与划分技术，多路复用用于固定通信网络中数据的复用解复用，多址接入用于移动通信网中都用户的合并和分离。
  2. 1-4G：OMA技术–正交多址接入技术。正交参量可以是标量、向量，满足正交性，接收端采用正交信号识别器识别各路信号。正交参量有频率FDMA（模拟制式是理想的，实际中模拟电路有非理想特征，需要引入保护间隔）、时间TDMA（考虑用户距离基站的远近，相对时延有大有小，会引入保护时隙）、码字CDMA（用户分配扩频码，扩频码码域之间相互正交）、SDMA空分多址（小区分裂，采用定向天线。3G中TD-SCDMA中智能天线的方向可以改动以实现空分效果，4G中MU-MIMO基站端通过波束成形的方式实现空分。）以上内容为PHY形式的固定式的多址接入方案，MAC层也有随机多址方案，典型技术有ALOHA技术、slot-ALOHA技术（2G-5G中使用）
  3. CDMA的两种方式（1）DS-CDMA：将窄带发送信号与高速的扩频信号相乘，信号带宽变宽，商业中常用（2）HF：跳频，用户在随机序列的控制下占用不同的时频单元，实现了整个信号的频段扩展–军事通信中常用
  4. 5G：NOMA技术

移动通信中的典型多址接入方式

  1. FDMA：TACS（fdd）上行890-915MHz，下行935-960MHz，25MHZ频段划分为1000个频道，各个频道25KHz，1000代表极限容量–1个小区同时支持通信的最对的用户数目，实际系统在空间上划分可以复用。
  2. TDMA：GSM体制，1帧划分为8个时隙，首尾包含保护比特。总频点数*每个频点提供的时隙数=GSM提供的时分信道总数，GSM的硬容量也是1000.
  3. CDMA：CDMA2000，IS-95体制，容量为55*20=1100，使用15阶的m序列构成的伪随机来产生
  4. OFDMA：正交形式的FDMA

码分多址CDMA中的地址吗

  1. 信道标识码、基站标识码、用户标识码都需要进行标识
  2. GOLD码、M序列是常用的伪随机序列

伪随机序列PN和扩频码的理论基础和分析

CDMA的两种典型方式：DS-CDMA直接扩频，HF–跳频。

chapter5 鉴权和信息安全

概述

  1. 空中接口信息安全问题：窃听、假冒、重放、数据完整性侵犯、业务流分析、跟踪（用户位置、状态）
  2. 网络侧数数据安全问题：网络内部攻击、数据库非法访问、对业务的否认

保密学的基本原理

  1. 经典密码学1949年（密码设计，防止窃听–对称的密钥体制）、现代密码学1976年（双钥体制–加密、解密密钥不一样，防止对信息的篡改）
  2. 现代密码学中发端K1是公开的密钥，接收端K2是私钥–双钥体制
  3. 经典密码学中两大编码的密码分支–序列密码、分组密码。序列密码对流进行加密，比特依次加密；分组密码对分组进行加密变化。
  4. 一次一密是最安全的–元首的通信，使用纯随机序列进行加密
  5. 常见的序列加密方法：对m序列进行RC算法的非线性变化。常见的分组加密方法：映射关系就是密钥，代表的算法–DES算法、Kasumi算法、AES算法
  6. 现代密码学：公开密钥依赖算法复杂性问题（单向陷门函数）–RSA算法

2G系统的信息安全

GSM系统的鉴权和加密

  1. GSM的安全保密性能：防止未授权的非法用户接入、防止空中接口非法用户窃听的加解密技术（采用序列加密的方法）、防止非法用户窃取身份吗和位置信息的临时移动用户身份吗TMSi更新技术（表示用户身份–IMSI国际移动用户识别码32bit，空口寻找IMSI代理TMSI–不断更新）、防止未经登记的非法用户接入和防止非法用户过期终端在网中继续使用的设备认证技术（网络测配备用户设备识别寄存器EIR，存储每个手机唯一的IMEI–合法的设备）
  2. GSM的漏洞：sim/ms的接口翻录–接口之间没有标准化，A3/A8鉴权算法的破解（Comp128算法机制，破解获得ki）、A5算法漏洞–采用序列密码加密防止窃听（a5/0中国–不加密–不要使用移动电话讨论国家机密，a5/1欧洲，a5/2北美使用），sim卡攻击，网络伪装攻击（只存在网络认证用户，若存在伪基站可以上下行发送一切合法的信令），网络数据文传输（GSM中网络侧数据是明文传输的）

IS-95系统的鉴权和加密

  1. IS-95系统的鉴权和加密细节和GSM大同小异。IS-95系统的鉴权序列–SSD（128bit的半永久性共享加密数据）：移动电子序号ESN、鉴权密钥A钥。ESN功能类似于IMSI码–用户识别码41阶的m序列的掩码初始化（身份识别、加密的双重功能）。

3G系统的信息安全

  1. WCDMA系统采用双向认证技术（网络和用户之间都需要认证，避免了伪基站的存在），建立了完整的认证与密钥协商机制(AKA)，实现数据加密防止窃听和数据完整防止篡改的双重保护
  2. WCDMA的加密算法主要有f8算法（加密算法）、f9算法（数据完整性保护）。f8、f9均采用kasumi算法。数据分为业务数据（主要进行加密）、信令数据（加密+完整性保护）。
  3. CDMA2000与WCDMA的加密机制类似。

B3G和4G系统的信息安全

  1. 4G中LTE与WCDMA类似，采用AKA机制，主要特点是密钥管理更加完善，多级进行密钥的管理和分发。
  2. WLAN无线局域网常用的有wifi，802.11机制中引入了WEP安全协议，错误使用了RC4算法，漏洞–加密算法漏洞、数据完整性缺陷（使用crc检测，数据完整性检测完全无效）。802.11进行改进得到802.11i、wapi协议，实际商用中两种安全机制都支持。
