part1 算命–紫微斗数

四化星–主力星，科、权、禄、忌

  1. 斗数中有四化星–主力星，科、权、禄、忌。批字的时候是流年在动，星都不动，星性最重要。
  2. 科：（1）科名：参加考试，写作文笔，画画很好，当老师也是科（2）名气很大：命宫里有科，有的人50岁之后有科–声名远播，90岁之后有科–死了之后才有名字。
  3. 科-相：手掌上面，有通天的天柱纹，主的就是科
  4. 权：（1）官印（2）性刚：个性刚强，不同位置上发生不同事情。夫妻宫有权星–性刚（另一半领导力强），命宫化权–天生的领导者，哪颗星开始化权–代表适合做生意还是官场
  5. 有科没有权，只能做军事，不能做领导。相–眉毛短短一点–做参谋
  6. 权–相：（1）颧骨来拿到耳朵后面，主管权，颧一定要有肉，最忌讳皮骨相连–孤颧。（2）犀牛骨：眉毛的骨头外凸–武将、军人、警察、司法官有权
  7. 禄：（1）财（2）性守–命宫化禄：守财奴，一毛不拔；禄在父母宫–父母从商，禄在财帛宫–自己会做生意，禄在命宫–先天有祖业到身上
  8. 禄–相：男人看鼻，鼻子代表财帛宫–事业，两种鼻子最好：狮鼻、悬胆鼻。女人鼻子主夫妻，女人手相–枯干
    狮鼻–鼻子很大、有肉

    悬胆鼻–鼻子细长悬胆
  9. 化忌（1）劫杀–意外横祸、官司纠纷、破财（2）想不开
  10. 化忌脸相–五官紧缩在一起–想不开–念行命。耳朵比眼睛高，眉毛在额头和眼睛中间–非富即贵–大开大合，做事情有担当有魄力。
  11. 算命要检讨自己，帮助别人。念影响行为，行为决定命运，悟道之后需要坚心。
  12. 气色黄润色，主喜事，结婚上尤其在鼻子、眼角很黄，额头上有黄–升官。化忌色–黑暗、灰尘。死色–将死之人的气色–枯萎色、青蓝色，头发上没有油、皮肤不反光，人好好的坐着，警惕下一个节气到来–大凶。
  13. 24个节气，一个节气两个礼拜：立春、雨水、惊蛰、春分、清明、谷雨、立夏、小满 、芒种、夏至、小暑、大暑、立秋、处暑、白露、秋分、寒露、霜降、立冬、小雪、大雪、冬至、小寒、大寒。

南斗

天府星

巨门星
  1. 巨门星本身是一个凶星，并不是吉星，代表口舌、是非、官司、牢狱之灾。
  2. “庙旺利陷”这个概念，它是用来形容一个星座的亮度的，亮度大比如一颗星是庙或者旺的话那么这颗星相对就比较好，如果是“利”或者“陷”的话，这个星座本身的一些好的特性也会打折扣。
  3. 如果巨门星是庙或者旺，也就是亮度比较高的话，它的灾就要去掉大半，此时的巨门星代表能言善道，口才非常好；那如果巨门星是落或者陷，也就是亮度不怎么亮的情况下，更多就代表口舌、是非、官司、牢狱之灾。
  4. 巨门，顾名思义，门很大嘛，那学校的门也很大，一个庙旺的巨门星可能代表学校的校长；但是落险的巨门星可能进了监狱的大门，只不过这个大门是关闭的，坐在牢里。所以同样一颗星，亮度不同情况也不一样。
  5. 巨门星入命的人有两种相貌，一种体型个子很小，说话音量很高，也就是型小音高量大；还有一种长得长绒肥胖，也就是身体很长，有体毛，毛发很多，比较肥胖。

六杀星

  1. 羊、陀、火、铃、空、劫，实际上它们的全称是：擎羊、陀螺、火星、铃星、天空、地劫。六杀星在命宫、有人的宫–解释不一样，在没有人的宫解释也不一样。
  2. 星辰的亮度：六杀星根据星座亮度也就是庙、旺、利、陷的不同。吉星在庙旺的时候都表现出比较好的特性，在利的时候表现平平，在陷的时候最差，碰到杀星同时又落陷那就是差中之差了。
  3. 如果遇到了陀螺星那本身是不好的，但是同时碰到了显示为庙旺的火星在同一个宫位，那么这个杀星就会变成吉星。我们有两个从人事方面总结得出的经验需要知道，叫做“吉处藏凶必凶”和“凶处藏吉平安”。例如看到都是吉星的时候里面还有一颗杀星落陷在里面，这个叫做吉处藏凶；另一种，看到的都是杀星，但还有一颗庙旺的杀星在里面，这就叫做凶处藏吉，反而是平安的。
  4. 举例：一群人趁着假期很开心的出去玩，这首先是吉事，但没想到路上旅行车出现事故导致了伤亡，这种事先预料不到到或者不想预料的凶事就容易在劫难逃，这是“吉处藏凶必凶”。相反的，一个人看到前方有火灾，这是遇到了危险，但正是因为他知道前面的路有危险所以肯定会绕开从而避开危险。这也像消防员冲向危险的火场，但我们很少听说消防员被火烧死的，因为他们事先就对危险的火场有认知，清楚会发生什么以及要怎么做，这就是所谓的“凶处藏吉平安”。
  5. 我们在实际解盘的时候，如果发现命盘的主人会碰到这种“吉处藏凶”的情况，那就会告诉他要知道“戒”，即“戒之初、戒之盛”。–戒要注意时机，保持谦卑。
    （1）“戒之初”是指在事情刚开始的时候要谨慎。比如前面那个出去旅游的例子，我们在出发之前，就应该了解到对应的接待的旅行社的情况，包括旅行社安排什么车辆，司机的技术如何，有没有酒驾醉驾的记录，投诉记录如何等等，统统明了以后再开始行动。这也像和别人签合同之前把该避免的情况都统统写上，先小人后君子再行事。这就是“戒之初”。
    （2）“戒之盛”是指越在最好的时候，越要警惕。比如自己做生意的老板，在生意越好的时候，越可能觉得自己无所不能从而做出错误的投资，进入自己不熟悉或者超过自己能力的项目当中，从而导致亏损。这也是意气风发时候容易栽跟头的一种表现。为什么要戒在最好的时候呢？一个欠款几十万的流浪汉睡在路边，小偷都不会想打他的主意，但是一个人意气风大的时候，越容易觉得自己流年那么好何必警惕呢。所以如果看到流年宫位统统是吉，但是里面藏着一个落陷的杀星，就容易出问题了。所以要告诉客户越是这时候越要警惕。
擎羊、陀螺–需要目所无事
  1. 陀螺、擎羊代表有小人出现而且还看不到小人，往往这时候小人就在你身边。一般来看越了解你的人才越能害到你嘛，不认识你的人根本害不到你。
  2. 陀螺、擎羊看不到小人这一点，还可以看做你从外表看很正常，但是不知道里面出了问题，所以这两颗星也代表身体外表看上去正常，但是里面有问题，也就是身体有疾病。所以流年碰到这两颗星还要小心身体健康方面的问题–开刀见血光。
  3. 擎羊本身代表刀刃、见血光，那如果这是一个庙旺的擎羊星，那就代表说这个刀下去以后平安无事，比如手术以后就没有后续问题了。但是没有事不代表什么都不用注意了，还是要保持警惕。
  4. 相：左边眉毛有纹路上去，小人之相。
  5. 前面讲了杀星庙旺的情况，对应的，杀星遇到落陷变得凶上加凶，因为它来阴的，除非知道“戒”，不然很难防范，比如人事方面遇到落陷很可怕，知人知面不知心，来阴的很难防范得了。
火星、铃星
  1. 流年碰到火星、铃星，代表可能会遇到火灾，或者个性上有冲突，
  2. 比如跟朋友做生意吵架散伙、跟朋友分手等冲突很大的事件。
  3. 比如刚好这两颗星在夫妻宫，那就代表说夫妻两个人大打出手；
  4. 如果进一步的同一个宫内还有巨门星，结果呢就表示夫妻两最后还要去法院互告。
天空、地劫

天空、地劫这两颗星，顾名思义代表意外、耗散。比如一个做生意的人流年宫位碰到天空、地劫，他这一年肯定要赔钱。

part1 算命–紫微斗数

  1. 命中有很多格，讲重要的格。
  2. 总格：
    （1）日月夹财：寅宫（太阳）、丑宫（命宫）、子宫（太阴）–日月夹财：一世财路很大，即使命宫没有主星
    （2）紫薇天府相夹，掌权

副星

天魁星、天钺星–科甲星

  1. 天魁星、天钺星相夹为贵
  2. 羊陀夹杀–累，家里面外面都是小人，命中的小人很多，真理说才对，领悟后就没有了。
  3. 如果一个人的命盘里面，文昌、文曲、天魁星、天钺星在三方四正里面都会到，那么即使这个长得再粗鲁、再壮硕，考试一定很厉害。所以我们不要被外形所蒙蔽迷惑。生活中常见到这样的例子，一个人长得清清秀秀就觉得这个人读书肯定厉害，一个人外形和语言都粗鲁就觉得这是个俗人，其实不是这样的。
  4. 手上有天柱纹–科甲厉害的人

曲昌：文曲星、文昌星–科甲星

  1. 文曲星代表才艺、博学多能。文曲星入命宫的人看起来比较斯文，文绉绉的。文昌星代表的是科甲，读书、考试很厉害，流年碰到文昌就是金榜题名。
  2. 如果只有文昌入命宫而文曲甚至都没在三方四正上会到（遇到）的话，那这个人就会变成书呆子。什么是书呆子呢？就是学而不思则罔的人，懂得很多，但联系不到实际上来的人，容易被骗。
  3. 如果文曲文昌同时会到，那就变成文采风流。这种情况对于男孩子来说很好，才子嘛；但是女孩子会到这两颗星不好，会变成女孩子的桃花运。
  4. 我们在算命时候，不要跟女孩子说“你有桃花”、或者“你这个是桃花命”这种话，合理的方式有很多，比如“男孩子看到你就会很喜欢你”等等。
  5. 现在来做一个简单的推理，如果一个人34岁的流年宫位中，刚好逢到文昌星，结果这颗文昌星显示化忌，代表这个人这一年有考试但是不中。
  6. 紫微斗数的很多结论都是这样加加减减类推出来的，看起来我们现在讲的好像很零散，但是以后具体来看命盘的时候，把这些知识概念合起来，推导就很简单了。

红鸾星、天喜星–喜庆

  1. 中国人会说一句成语叫做红鸾星动了，就是说有婚姻喜事将近。其实这个成语就来自于紫微斗数。
  2. 男人命宫里有红鸾，也就是命带红鸾，必招美妻；女命红鸾，必有贵夫。
    (1)在实践中，命带红鸾的男孩子很奇特，漂亮的女孩子才会去追，不漂亮的连看都不看，看到漂亮的以后无法忍耐不追不行。
    (2)命带红鸾的女孩子同样奇特，一般来说不漂亮，她们的脑筋很清楚，看的很明白，自然而然的知道谁是贵夫应该嫁给谁，而且她们还可以看到嫁给先生的时候先生没有什么但是以后会有。当然这种能力在易经中也能学会，所以命里没有红鸾也能学。
  3. 一些其他流派说，必须要会到红鸾星、天喜星才可以结婚，其实不然。有很多其他情况也会结婚，例如流年宫位刚好是夫妻宫有可能结婚，流年宫位刚好是父母宫奉父母之命媒妁之言可能会结婚，所以有人的宫位都可能会结婚；甚至没人的宫位也可能会结婚，例如流年宫位是财帛宫，那就有女孩子在这一年嫌在公司帮人做事不开心，于是辞职自己做生意，然后第一个签约的客户就是她未来老公这种例子。所以婚姻在什么时候不仅仅看红鸾天喜。至于具体在什么时候，甚至如何人为的去干预影响，这些我们以后看具体命盘的时候又讲解。

左辅星、右弼星

左辅星、右弼星辅助紫微星的，只要看到紫微星出现的时候如果有左辅右弼来会，这个紫微星就会变成诸侯君王之象。

三台星、八座星。

  1. 与左辅右弼跟紫微星的关系类似，南斗星君天府星同样有自己的左膀右臂，三台星、八座星就是辅助天府星的星座。
  2. 如果天府星有三台、八座，就像南斗星君有了文武百官，这个时候天府星就有了解厄制化的特性。但是北斗星君紫微星要比南斗星君天府星强，因为紫微星即使没有左辅右弼，它都可以解厄制化，而天府星一定要逢到三台八座才有这种能力，否则单独一颗天府星是不能解厄制化的。

十年大运

命盘图

  1. 闰月生人做下月论：农历四月，刚好是闰四月，排到五月
  2. 子时生人：11-12-1，11-12晚子时（根据前一天来），12-1早子时（日子要换，换成第二天的）。
  3. 命好不如限好，命里都是吉星，代表先天好，十年大运–限，流年好很好。
  4. 小儿善知人意，主夭；小孩子有大人的样子，夭折，限在七岁以前，没有小孩子的天真活泼，主夭折。小孩子要有小孩子的天真，童蒙。需要教小孩子不会的，带他去玩，培养胆量。
  5. 命由水二局、木三局、金四局、土五局、火六局–排的时候就有了，没有代表的意义，只是用来排局定星座，不用于告知从事的职业。
  6. 批八字先批十年，总运批好，这是最难的。十年为主，1年的小流年为辅，来辅佐。比如十年是财星，小流年是天空地劫–这一年是赔钱的，但十年是赚钱的。
  7. 小流年不需要看左右夹，只需要看对宫和本宫。小流年从戌宫开始起子，比如43年生是甲午年，民国83年，今年流年在哪个宫：83-43+1=41岁，41岁从戌宫起子，逆时针：子丑寅卯、辰巳午未，申酉戌亥，对应到戌、未、辰、丑的宫位。甲午年生，午对应到辰宫的位置，此为1岁，男顺女逆，回到辰宫是13岁、25岁、37岁…，41岁对应到未宫的位置。
  8. 左边四个宫位：从下向上–春夏秋冬，春分、夏至、秋分、冬至这四个节气，前一日为离日（木火金水四个节令分离的时候，气最弱的时候最危险），立春、立夏、立秋、立冬前一日为绝日（春夏秋冬节日交换，向下坠落，大凶）。举例：12月25立冬，12月17面色枯槁，流年六杀，主星进来的时候，杀星还落陷，百分百死定。这八天一定要避开，做任何事情都要避开–人气最弱的时候。举例：夏季的时候好、秋季的时候凶，化忌压力在这边、立秋之前就会出事。

禄存星

  1. 禄存的特性类似之前学过的四化星中的“化禄”，指的是财禄，所以它是一颗财星。
  2. 禄存星入命的人，性格上会显得小气。所以如果你的儿子或者女儿命中有禄存星，那很可能你给的零钱压岁钱他会统统藏起来，然后哪天出去玩花的也是你的钱，自己的钱都自己好好收着。
  3. 流年宫位如果有禄存星在的话，那财运会很旺，这一点等同于流年宫位刚好是财帛宫财气旺一样，事业上都会赚钱。
  4. 有一种特殊的格局，叫做“禄马交驰”，是指禄存星或者化禄星在三方四正中逢到另外一颗叫做天马星的星辰，这时候在事业上会进大财。但是禄马交驰的钱一定是很辛苦赚来的，毕竟顾名思义就有奔波的意象嘛。
  5. 一般人有禄马交驰就不得了了，但还有一种更特殊的情况，就是化禄星、禄存星、天马星在同宫，那就是巨富。如果一个人生下来就禄马交驰在命宫，同时逢到巨门星入庙，那这个人生下来就在豪门世家里面，因为巨富的人再逢到代表豪门宅邸的入庙巨门星，肯定是非常不得了。

part2易经–六十四卦–卦有解卦

乾为天

人间道

  1. 初爻、二爻、三爻、四爻、五爻、六爻–事情的发展是六个阶段。进入各个阶段层次不一样，下、将、相、宰、君、过–六爻的层级。
  2. 初爻是下爻，潜龙勿用：下–下明-闲云野鹤；进行自修。
  3. 二爻是将，当上了主管，县长–见龙在田。和下爻的心一样，若是阳，就主刚，本心不变，适合交对你有意的朋友–懂得比你多的人、术业有专攻的同样类型的人
  4. 三爻是下卦–内卦的顶，终日乾乾，需要警惕自己，进一步进德修业，为下一步做准备，内心的东西不变，仍向下看。
  5. 四爻，君王的左右手，需要或跃在渊，待时机而动，顺势而动–观看君王的动作–时局很重要。出现政治分权，政争时最好跑路，不要参与，再从国外回来。
  6. 五爻，变成君王–飞龙在天–坚心：始终如一，需要往下看。
  7. 六爻，还是往上看，亢龙有悔，过激，此时谁也不信。无民无辅–例如袁世凯。
  8. 所有阶段都一定向下看，按照这个方式来做就是乾为天，让自己、别人都可以变成卦。别人处在哪个卦的第几爻，往上走应该怎么做，变成地天泰等等–改变念/想法
  9. 易经依靠天地定位

占卜

  1. 楼梯上有怪人在上面：棺。
  2. 人间道来说：官运不亨通。官人站在梯上，居上位而不骄，代表现在骄，头往上看不好，往下看就会升官。
  3. 空中有禄，代表空想，断念才好。
  4. 图上有工匠，改门面，改朝换代。
  5. 玉在石头中：事业还没有成功。
  6. 卜疾厄，代表凶；卜生男育女，代表六个阳代表阳。
  7. 人雕刻玉石：需要小心谨慎。
  8. 卜婚姻：婚姻可成，但是需要艰辛。
  9. 月在当空：清明的象，代表政治清明，你是错的，你本身太过于骄。
  10. 部长、组委是相、院长是宰，科长处长是将。
  11. 父亲居住母位：天卦

坤为地

人间道

  1. 母亲居住母位：地卦。母亲住到父亲的位置–地天泰。
  2. 全部是阴爻。
  3. 坤为地，乾为天，德同法异，始终如一，坤主柔静。
  4. 地能载万物，接受一切，再返还给天，不邀功。
  5. 一、三、五是阳位，二、四、六是阴位。第一爻应该是阳，但是坤是阴–柔（寒）：履霜，坚冰至：代表发现小人的时候，赶快绝交–戒之初；强烈的修正。去一个公司，一开始就是阴，履霜，坚冰至，赶快退出。
  6. 二爻居阴位：直方乃大。直–直言，方–道义，关于这些品德，即使不懂什么其他知识，也对你没有害。
  7. 三爻阳位，阴的性–柔，具有含章之美，天经地义，在乎做了多少事情。
  8. 四爻–闭口，挂囊。四爻和五爻最近，君臣、老板、天子、董事长与你意见不合，闭口去做，否则就被误解、功高震主。
  9. 五爻，军位，阴居君位–女人做皇帝，黄裳元吉：下等的衣服（黄裳），元吉–对大臣谦逊，本身阳位不要去亢，否则进入六爻
  10. 六爻，龙战于野，其血玄黄。天是杂色，一定会有不好的结果。
  11. 女生是董事长，这个时候不要去抗，需要穿着朴素，听取意见，保持谦卑。
  12. 三爻对应经理的位置，需要含章之美，不要争功、加薪，很快进入第四位。

占卜

  1. 人坐在椅子上、看台上–这是官人，两串钱–忧心忡忡，贪污–想拿又不想拿
  2. 十一口：吉、陈，众口铄金，也代表替代众人发言的相
  3. 人–民心，代表有人检举
  4. 马–属马的人–肖马，指官贵，指调动。属马的人逢此卦调动。
  5. 图上有四点：黑也，还没有拨云见日。
  6. 司法界的人事，属马，办贪污的案子–卜卦：坤为低。吉的意向在，但是有马，代表可能会离开交给别人。
  7. 占卜选举，得此卦一定上位，为百姓发言。
  8. 金价神人–得民心的人。
  9. 堠：土墩堆起来的东西，授官封侯的相。会不会升将军，占卜为此卦，姓陈、马上位，姓朱则回头，上不了位。
  10. 生男女：坤地，第几胎？6。还是女儿。
